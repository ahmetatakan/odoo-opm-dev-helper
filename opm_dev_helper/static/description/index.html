OPM Dev Helper

Hot-reload for faster Odoo development

Requires: Odoo Plugin Manager (CLI)

  • Instant hot-reload after code or XML changes
  • One-click cache flush (QWeb & View cache)
  • Quick module install / upgrade helpers
